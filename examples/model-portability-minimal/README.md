# Model Portability Minimal

This example records synthetic evidence for comparing two neural model backends
against the same Character Layer profile, task set and behavior fixtures.

Validate:

```bash
node packages/cli/validate-mirai-graph.js model-portability-evidence examples/model-portability-minimal/results/model-portability-evidence.json
```

## Boundary

The example is synthetic evidence only. It demonstrates the measurement shape;
it does not prove that any real model can replace another model in production.
