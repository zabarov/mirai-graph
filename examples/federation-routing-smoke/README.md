# Federation Routing Smoke Example

This synthetic example shows the public GrowGraph routing control pattern:

```text
request -> route -> explanation -> evidence -> regression -> health dashboard -> governed learning proposal
```

The example uses generic capability names only:

- `implementation`
- `deployment_safety`
- `quality_assurance`
- `research_review`
- `content`
- `search_visibility`

It is not based on private federation data and does not authorize runtime
execution or canonical graph updates.
