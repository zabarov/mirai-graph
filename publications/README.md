# Publications

This directory tracks public publications, preprints, citation metadata and
reproducibility notes related to GrowGraph.

## Current Status

The model is being prepared as an open research and standardization project.
Publication links should be added only when the public page, DOI or venue record
is confirmed.

Do not describe a preprint as peer reviewed unless the venue record explicitly
states peer-reviewed publication status.

For the current public evidence framing, see
`docs/research/scientific-evidence-package.md`.
