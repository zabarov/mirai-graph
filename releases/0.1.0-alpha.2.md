# Mirai Graph 0.1.0-alpha.2

Release type: context selection and profile documentation alpha

## Summary

This release improves the alpha context-pack generator and documents the first
project-management profile lessons from independent implementation pilot 001.

## Added

- Token matching and one-hop relation expansion for context-pack generation.
- Updated generated context-pack artifacts.
- Project-management profile documentation.
- Pilot report and adoption guide wording updated to reflect alpha context
  selection instead of full-graph-only context generation.

## Validation

Run:

```bash
npm run release:check
```

Expected:

- all existing package, seed and profile validation checks pass;
- context generation succeeds;
- synthetic benchmark remains reproducible;
- negative fixture fails as expected.

## Important Limitations

- Context selection is heuristic.
- Selection explanations are not yet detailed enough for semantic review.
- Semantic completeness must still be reviewed separately.
