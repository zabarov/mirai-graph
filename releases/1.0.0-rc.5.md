# Mirai Graph 1.0.0-rc.5

Status: release-candidate
Date: 2026-06-27

## Summary

`1.0.0-rc.5` adds Semantic Intent Resolution as an input-control extension
contract.

The contract turns a free-form request into a governed decision before
execution starts:

```text
request
-> intent hypothesis
-> work scale
-> owner/process decision
-> context-slot check
-> clarification, brief, workflow restore or safe execution
```

This is not a new standalone profile and not a knowledge-storage layer. It is a
control boundary that helps projects, organizations and AI employees avoid
starting the wrong process from an ambiguous request.

## Claim Boundary

This release validates the public shape and safety boundaries of Semantic
Intent Resolution. It does not prove that AI agents always choose the correct
process, does not authorize live or destructive actions, does not update
canonical graph state from generated briefs or proposals and does not prove
production readiness.
