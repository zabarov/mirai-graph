# Mirai Graph v0.1.0-alpha.5

Status: release notes

## Theme

Public standardization of graph-managed skill runtime patterns.

## Included Since v0.1.0-alpha.4

- Hybrid Source Of Truth standard draft.
- Mirai Graph Adoption Levels standard draft with `GGA0..GGA10`.
- Runtime Enablement standard draft for task-scoped graph-mode use.
- Projection Views standard draft.
- Alpha `skill_runtime` profile.
- Minimal synthetic skill-runtime fixture.
- Profile conformance result for the skill-runtime fixture.
- Public-safe transfer note for reusable lessons from graph-managed skills.
- README and profile fixture documentation updates.

## Validation

Validated with:

```bash
npm run release:check
```

The release also passed the public safety scan used by maintainers to check for
private paths, obvious secrets and unsupported claims.

## Claim Boundary

This release should claim only:

```text
Mirai Graph v0.1.0-alpha.5 adds public alpha standard sections and an executable
skill-runtime profile for governed capability graphs.
```

It should not claim:

- graph-only runtime readiness;
- semantic completeness;
- operational effectiveness;
- runtime safety;
- universal validity;
- publication of private skill internals.
