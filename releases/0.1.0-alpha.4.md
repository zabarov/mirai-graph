# GrowGraph v0.1.0-alpha.4 Draft Scope

Status: draft release scope

## Theme

Executable standard hardening plus the first public AI employee profile.

## Included Since v0.1.0-alpha.3

- Context-pack selection explanations with object relevance scores and relation
  inclusion reasons.
- Context-pack validation mode for generated pack metadata, graph references
  and explanation coverage.
- Negative fixture for missing context-pack selection explanations.
- Positive profile conformance result artifacts.
- Public capability map from practical graph-method areas to GrowGraph
  standard, schema, tooling, profile and research artifacts.
- Alpha `ai_employee` profile.
- Graph-as-AI-brain standard draft.
- Action/runtime boundary standard draft.
- Feedback learning gate standard draft.
- Minimal synthetic AI employee fixture.
- AI employee adoption guide.

## Validation Target

Before publishing the release:

```bash
npm run release:check
```

The release should also pass the public safety scan used by maintainers to
check for private paths, secrets and unsupported claims.

## Claim Boundary

This release should claim only:

```text
GrowGraph v0.1.0-alpha.4 improves executable alpha validation and introduces
the first public AI employee graph profile with explicit action/runtime and
feedback-learning boundaries.
```

It should not claim:

- real-world AI employee safety;
- autonomous execution readiness;
- Larena/AI runtime conformance;
- productivity improvement;
- external empirical validation.
