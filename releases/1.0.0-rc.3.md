# Mirai Graph v1.0.0-rc.3

Status: release candidate prepared in repository; tag, GitHub Release and npm
publish require separate release execution.

## Theme

Developer-First Onboarding.

This release candidate makes the repository easier for an ordinary developer to
understand before reading the full standard. It explains the concrete software
project problems Mirai Graph solves, what can be done in 10 minutes, what files
are created and how graph structure helps AI-assisted development.

## Added

- Developer-focused guide:
  [Mirai Graph For Developers](../docs/adoption/for-developers.md).
- README sections for:
  - concrete project problems;
  - developer use cases;
  - current npm publication status;
  - created files and generated bootstrap proposal output;
  - how Mirai Graph helps AI assistants.

## Changed

- README now starts from developer value instead of abstract standard framing.
- Documentation map links to the developer entry point.
- Getting-started and 15-minute tutorial now point developers to the practical
  guide and state the current pre-npm publication boundary.
- Playground and active evidence docs are aligned to `1.0.0-rc.3`.
