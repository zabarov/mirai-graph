# GrowGraph v0.1.0-alpha.6

Status: release notes

## Theme

Public standardization of executable GrowGraph governance patterns.

This alpha started with explainable federation routing control-loop patterns and
now also packages the implementation-control and process-control layer needed
to make GrowGraph adoption inspectable with public-safe schemas, fixtures and
validators.

## Included Since v0.1.0-alpha.5

### Routing Control

- Route Explanation standard draft.
- Routing Fixtures standard draft with forbidden route shapes.
- Federation Health standard draft with conflict details.
- Route Regression standard draft.
- Public schemas for route explanation, routing fixtures, routing fixture run,
  federation health dashboard and route regression result.
- Synthetic federation routing smoke example with per-fixture route evidence,
  high-risk explanation, route regression and federation health dashboard.
- `validate:routing-control` validation command.
- Public-safe transfer note for federation routing control-loop lessons.

### Implementation And Process Control

- Implementation Control profile hardening.
- Implementation Control Cycles standard and synthetic example.
- Launch Record standard, schema, example and validator.
- Process Transition request schema, passing fixtures and negative fixtures.
- Process Control Contract standard, schema, example and validator.
- Work State Machine and Recovery/Resume schemas and examples.
- Terminal Kaizen closure rule for significant process transitions.
- Clear boundary between human/projection stages and executable state-machine
  movement.

### Governance Standard Modules

- Graph DNA Alignment standard, schema and synthetic example.
- Risk Control Matrix standard, schema and synthetic example.
- Multi-Agent Coordination standard, schema and synthetic example.
- Source Boundary Contract standard, schema and synthetic example.
- Organization Governance profile, standard draft and synthetic example.
- Public-safe project-management transfer note for implementation-control
  patterns.

### Developer Tooling

- Validation commands for implementation-control cycles, launch records,
  process transitions, process-control contracts, DNA alignment, work state,
  recovery, risk, coordination, source-boundary and organization-governance
  examples.
- Negative test coverage for invalid graph packages and invalid process
  transitions.

### Navigation And Evidence Packaging

- Role-based README navigation.
- End-to-end control-loop tutorial.
- Playground index for runnable CLI demos.
- Scientific evidence package framing synthetic evidence, pilot evidence and
  scientific claim boundaries.

## Validation


## Claim Boundary

This release should claim only:

```text
GrowGraph v0.1.0-alpha.6 adds public alpha standard and schema shapes for
explainable federation routing, implementation-control cycles, launch records,
process transitions, process-control contracts, governance boundaries and
public-safe evidence packaging.
```

It should not claim:

- private federation behavior;
- graph-only runtime readiness;
- runtime safety;
- owner-source rewrite permission;
- scientific proof of effectiveness;
- production readiness for autonomous execution;
- automatic canonical update from generated context, feedback, evidence or
  proposals.
