# Mirai Graph v1.0.0-rc.1

Status: release candidate prepared in repository; tag, GitHub Release and npm
publish require separate release execution.

## Theme

1.0 Release Candidate.

This release candidate consolidates the public Mirai Graph standard from the
alpha line into a single verifiable package for external review. It does not
change the core claim boundary: Mirai Graph validates graph-managed structure,
evidence, process state and governance gates; it does not execute production
actions or authorize canonical updates by itself.

## Claim Boundary

This release candidate supports the claim that Mirai Graph has a coherent,
validator-backed public standard shape for graph-managed knowledge, process,
AI-employee behavior, implementation control, instrumentation and evidence.

It does not claim:

- peer-reviewed scientific proof;
- broad external validity;
- production autonomous execution safety;
- model-backend equivalence;
- automatic canonical updates from generated context, evidence, feedback,
  cockpit metrics or proposals;
- npm package publication until npm release-state checks confirm publication.
