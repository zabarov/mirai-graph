# Mirai Graph 0.1.0-alpha.0

Release type: initial public foundation

## Summary

This alpha release establishes the public Mirai Graph repository as a starting
point for standardization, developer adoption and reproducible synthetic
examples.

## Included

- Public README and roadmap.
- Repository purpose brief.
- Concept and terminology docs.
- Initial standard draft.
- Object and relation models.
- Lifecycle/readiness, evidence/provenance, governance and conformance sections.
- Context-pack generation, profiles and test-suite direction.
- JSON schemas for core package elements.
- CLI validator.
- Minimal graph example.
- Synthetic context-reduction benchmark.
- Initial adoption and contribution guides.

## Validation

Run:

```bash
npm test
```

Expected:

- minimal graph validates;
- synthetic benchmark validates;
- synthetic context-reduction calculation returns `56.4516%`.

## Important Limitations

- This is an alpha foundation, not a stable standard.
- Synthetic benchmark evidence is not external validation.
- The project does not yet include an independent implementation pilot.
- Context-pack generation has a documented model but no automated generator.
