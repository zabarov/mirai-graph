# Mirai Graph v0.1.0-alpha.9

Status: release preparation notes

## Theme

Public rename from GrowGraph to Mirai Graph.

This alpha prepares the public standard, package and CLI identity for the
`mirai-graph` slug while preserving temporary GrowGraph compatibility for
existing alpha users.

## Included Since v0.1.0-alpha.8

### Rename Transition

- Public standard name is now Mirai Graph.
- Package name is now `mirai-graph`.
- Primary CLI entrypoint is now `packages/cli/mirai-graph.js`.
- Preferred package manifest is now `mirai-graph-package.json`.
- GrowGraph remains documented only as the former working name and as a
  temporary compatibility alias.

### Compatibility

- Legacy `packages/cli/growgraph.js` delegates to the Mirai Graph CLI.
- Legacy `packages/cli/validate-growgraph.js` delegates to the Mirai Graph
  validator.
- Validators accept both `mirai-graph-package.json` and legacy
  `growgraph-package.json`.
- `npm test` includes a compatibility smoke check for the new and legacy CLI
  paths plus the legacy manifest fallback.

### Documentation

- Added a transition guide:
  `docs/adoption/mirai-graph-transition.md`.
- Updated README, CLI docs, citation metadata, changelog and release navigation
  for Mirai Graph naming.

## Validation

Validate with:

```bash
npm run validate:mirai-graph-compat
npm test
npm run release:check
```

## Claim Boundary

This release should claim only:

```text
Mirai Graph v0.1.0-alpha.9 updates public naming and preserves alpha
compatibility for existing GrowGraph users.
```

It should not claim:

- a GitHub repository rename has already happened;
- schema namespace ownership has been finalized;
- npm package publication has happened;
- old evidence payloads were regenerated;
- peer-reviewed proof of effectiveness.

## Known Limits

- GitHub repository rename, push, tag and npm publish are not part of this
  local preparation batch.
- Schema `$id` namespace migration remains pending domain confirmation.
- GrowGraph aliases are transitional and should not be used for new public
  materials.
