# GrowGraph 0.1.0-alpha.1

Release type: standard, tooling and pilot evidence alpha

## Summary

This release turns the initial public foundation into a more usable
research-engineering alpha. It adds graph seed/embryo methods, transfer mapping
from the practical `$graph` skill, alpha tooling, profiles, research protocols
and the first independent implementation pilot.

## Added

- Practice-and-science program framing.
- `$graph` to GrowGraph transfer map.
- Standard gap report.
- Graph DNA, Graph Seed and Graph Embryo standard sections.
- Graph seed and graph embryo schemas.
- Minimal graph seed example.
- Seed preview CLI command.
- Readiness score CLI command.
- Context-pack generator alpha command.
- Profile validation command.
- Context-pack, profile and readiness score schemas.
- Software specification and project management profiles.
- Measure-convert-optimize benchmark protocol.
- Semantic completeness review protocol.
- Independent implementation pilot plan.
- Independent implementation pilot 001 for synthetic conference planning.
- Worked tutorial from graph seed to validated package.
- Negative validation fixtures.
- `release:check` script.
- GitHub Actions CI workflow.

## Validation

Run:

```bash
npm run release:check
```

Expected:

- minimal graph validates;
- synthetic benchmark validates;
- graph seed validates;
- profiles validate;
- seed preview emits a graph embryo;
- context-pack generator emits a context pack;
- readiness score returns a capped pilot score for synthetic evidence;
- synthetic benchmark returns `56.4516%`;
- negative fixture fails as expected;
- whitespace check passes.

## Important Limitations

- Context-pack generation is still alpha and includes the full package graph.
- Readiness scoring is a navigation signal, not approval.
- Pilot 001 is synthetic; it validates package creation, not real-world
  productivity.
- Broader scientific claims require current-mode vs graph-mode comparisons,
  semantic completeness reviews and cross-domain replication.
