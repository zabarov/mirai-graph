# Mirai Graph 1.0.0-rc.4

Status: release-candidate
Date: 2026-06-16

## Summary

`1.0.0-rc.4` adds the proposal/experimental Dynamic Episode Layer. This layer
records how a concrete event moves through a Mirai Graph-governed process:
which objects, relations, policies, gates and technology steps activate, which
path is selected, which alternatives are blocked, what evidence is produced and
which findings should route to feedback, Kaizen or replay/regression.

The release keeps the central Mirai Graph boundary:

```text
episode trace = evidence
episode trace != authorization
episode trace != canonical graph update
```

## Claim Boundary

This release validates the public shape and safety boundaries of Dynamic Episode
tracing. It does not claim that real AI-assisted code generation drift is
reduced. That requires private or public pilot evidence with replay/regression
metrics.
